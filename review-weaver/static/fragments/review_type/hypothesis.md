# 综述类型：假设驱动

以用户核心假设为骨架，按假设归类文献，构建"共识→分歧→缺口"论证链。适用于有明确研究假设的实证论文文献综述。

## 论证链

`假设锚定 → 顶刊共识 → 分歧与原因 → 系统性缺口 → 本研究切入点`

## 结构规范

### 横向模块（80%-85%）

每条假设对应一个独立区块，内部按以下三层结构组织：

```
二级标题：假设 N：[核心关系简练表述]

1. 理论基础与共识（2-3段）
   - 核心理论框架引述（顶刊优先锚定）
   - 已形成的学术共识
   - 代表性实证证据（附评述）

2. 分歧与争议（1-2段）
   - 结论矛盾的具体表现
   - 分歧原因分析（样本/方法/测量/情境/理论视角）
   - 双方观点的期刊等级标注

3. 研究不足与缺口（1段）
   - 现有文献的系统性不足
   - 过渡句引出用户研究
```

### 纵向模块（15%-20%）

以时间或方法演进为线索，梳理该研究领域的发展轨迹：

```
二级标题：[研究领域]研究脉络与演进

1. 萌芽阶段（xxxx-xxxx）
   - 早期核心问题、代表性开创性文献、主要方法

2. 深化阶段（xxxx-xxxx）
   - 问题细化、方法论进步、新变量引入、机制讨论深化

3. 前沿方向（xxxx-至今）
   - 当前热点、方法论前沿、尚未充分探索的领域
   - 本研究在前沿脉络中的定位
```

## 起草规则

- 假设标题直接反映自变量→因变量的核心逻辑
- 每个假设下的文献按逻辑关系排列：基础理论→实证支撑→反面证据→研究缺口
- 评述段落采用"述—评"嵌套结构：先述文献结论，再评学术含义
- 分歧点是综述灵魂——无分歧讨论 = 无深度，评审会质疑批判性思维
- 过渡句先总结已有成果，再指出未解决问题，最后自然引出"本研究拟……"

## 假设标题示例

- `假设 1：人工智能通过提升创新效率促进企业高质量发展`
- `假设 2：数字化转型对绿色全要素生产率具有正向溢出效应`
- `假设 3：ESG 表现通过降低融资约束提升企业投资效率`
