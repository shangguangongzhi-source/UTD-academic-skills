# 默认立场（综述生成）

## 证据来源优先于一切

- 不要编造文献、结论、统计数字、方法论细节或期刊等级。
- 先构建论证逻辑，再写段落文字。
- 如果关键证据或边界缺失，写占位符并暴露缺口，不要填补。

## 综述不是文献列表

- 综述是**按论证逻辑组织的 synthesis**——按机制、方法、结论分组，而非按年份或作者罗列。
- 每段文献引述必须伴随评述：解读结论、比较研究、判断共识/分歧。
- 禁止"张三（2020）发现……；李四（2021）发现……"的罗列写法。

## 引用纪律

- 所有引用的文献**必须来自用户提供的精读报告或用户指定的文献清单**。
- 严禁新增未在输入中出现的文献。
- 引用格式：作者 + 年份。首次出现标注期刊等级。
- 英文文献 ≥3 人用 et al.，中文文献 ≥3 人用"等"。

## 顶刊观点前置

- 每个假设/主题下，**优先引述顶刊（UTD24/ABS 4*/经济学五大刊/NSFC A类）观点**。
- 顶刊观点作为讨论的"锚点"，其他文献围绕锚点展开补充或对比。
- 分歧讨论中，双方代表性文献均应标注期刊等级。

## 有限主张，校准动词

- 共识描述用强动词：`demonstrate / establish / consistently show`
- 分歧描述用中动词：`suggest / contrast / find mixed evidence`
- 缺口推断用弱动词：`remain unclear / potentially / warrant further investigation`
- 不使用 unsupported 的全称主张（"首次""前所未有""全面覆盖"），除非有直接证据。

## 输入要求 — 起草前必须确认

识别以下信息，缺失则暴露：

- **研究主题**：论文题目或综述主题
- **综述类型**：假设驱动 / 主题驱动 / 方法学综述（见 manifest 轴）
- **核心假设**（假设驱动型必需）：1-3 条，作为横向模块骨架
- **综述边界**：时间范围、学科范围、纳入标准
- **精读报告**：技能 2 产出的结构化报告
- **目标风格**：中文核心 / 英文 SSCI
- **篇幅要求**：默认 3000-5000 字综述主体
