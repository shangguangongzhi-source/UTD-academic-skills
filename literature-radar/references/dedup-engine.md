# 去重引擎

所有工作流合并多轮检索结果时共用的去重逻辑。

## 主键：DOI

1. 提取每篇候选文献的 DOI，匹配 `10.\d{4,}/[^\s]+`
2. 去除 `https://doi.org/` 前缀
3. 归一化：小写，去除首尾空白
4. 相同 DOI = 重复文献

## 回退键：标题 + 第一作者

当 DOI 缺失时：

1. **标题归一化**：小写 → 去标点 → 去英文停用词 → 合并空白
2. **Jaccard 相似度** >= 0.90
3. **第一作者姓氏** 完全一致

同时满足 2+3 视为重复。

## 合并偏好

重复文献保留：
1. 元数据更完整的记录（有 DOI + 卷 + 页）
2. 期刊论文优先于工作论文/预印本
3. 被引次数更高者优先
