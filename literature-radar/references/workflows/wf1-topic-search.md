# 工作流 1：按主题检索顶刊文献

**目的**：根据研究主题，在核心期刊池中精准检索高质量文献。

**使用**：[去重引擎](../dedup-engine.md)、[检索策略](../search-strategy.md)

## 流程

### 1. 理解检索需求

从用户输入中提取：
- **研究主题**（一句话概括核心问题）
- **核心关键词**（3-5 个，优先学术术语）
- **时间范围**（默认近 5 年）
- **语言偏好**（英文/中文/双语，默认双语）
- **学科方向**（经济学/管理学/金融学/其他，影响检索策略）

如果用户未提供完整信息，基于主题合理推断，不逐一追问。

### 2. 构建检索策略

参照 [检索策略](../search-strategy.md) 执行关键词扩展和查询构建。

多轮检索顺序：
- **第一轮**：核心关键词 + 顶刊名称 → 锁定核心池文献
- **第二轮**：扩展关键词 + 不限定期刊 → 发现有潜力的补充文献
- **第三轮**：被引追踪 → 对发现的高引文献追踪其参考文献和施引文献

### 3. 文献验证与分级

按 [search-rules.md](../../static/core/search-rules.md) 中的验证流程执行：
1. 真实性验证
2. 期刊分级
3. 相关性评估
4. 质量信号

### 4. 去重

使用 [去重引擎](../dedup-engine.md) 对多轮检索结果去重。

### 5. 输出

使用 `scripts/render_report.py` 生成 HTML 报告。回退方案参照 `static/templates/html-report.md`。
