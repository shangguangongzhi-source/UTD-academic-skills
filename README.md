# UTD Academic Skills

经管领域学术研究 TRAE Skills 工具集，覆盖文献检索→精读拆解→综述生成的完整工作流。

## Skills 概览

| Skill | 功能 | 触发词 |
|-------|------|--------|
| **literature-radar** | 顶刊文献精准检索与分级筛选 | 检索文献、搜文献、找论文、文献雷达、顶刊检索 |
| **paper-reader** | PDF文献结构化精读与8维度信息抽取 | 精读论文、读论文、拆解文献、论文解析 |
| **review-weaver** | 文献综述主体生成（横向分类+纵向脉络） | 写文献综述、观点整合、综述框架、观点织网 |

## 工作流

```
文献雷达（检索筛选）→ 精读拆解（8维度报告）→ 观点织网（综述生成）
```

## 架构

每个 Skill 采用 **静态/动态分层架构**：

- `SKILL.md` — 路由器，检测需求并加载匹配碎片
- `manifest.yaml` — 声明式清单，定义轴、碎片映射和按需参考
- `static/` — 版本化核心内容（期刊等级、搜索规则、工作流模板等）
- `references/` — 深度参考（去重引擎、检索策略、分歧分析等）
- `scripts/` — 渲染脚本（HTML 报告生成）

## 期刊覆盖

- **UTD24**（24本商管领域顶级期刊）
- **ABS 4***（扩展经济学顶刊，含经济学五大刊）
- **NSFC 管理科学 A 类**（22本中文管理学顶刊）
- **中文经济学顶刊**（经济研究、经济学季刊等）
- **第三梯队**：CSSCI、SSCI 二区（需标注理由）

## 安装

将对应 Skill 目录复制到 TRAE 的 `.trae/skills/` 目录下即可使用。
